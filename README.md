# Colin Christmas

Solo developer building **Tint Shop App** — a multi-tenant SaaS platform 
for the automotive window film industry. I own a tint shop. I know the 
domain from the inside, which means I'm building the product I actually 
needed and never had.

## Stack
Next.js 15 · Prisma · Neon · PostgreSQL · Clerk · MUI · Vercel

## How I Work
I use an agentic development workflow with Claude Code CLI as the primary 
executor — governed by a CLAUDE.md file, pre-merge security checks, and 
feature-branch discipline. I direct the agent, debug fault zones, make 
architectural decisions, debug and test the features.
